## 🖱️ Using the Interface

You see a small icon in your taskbar once you run the program. Click this icon to open the main menu. You can drag and drop your windows into specific slots on the canvas. 

The software remembers where you place each window. When you restart the program, it restores your workspace to the last saved state. You save time by avoiding manual window sorting. 

### Moving Windows
Grab the top bar of any window with your mouse. Drag it to a zone on your canvas. The window snaps into place automatically. This keeps your screen clear of clutter.

### Saving Layouts
Click the File menu at the top of the interface. Select Save Layout from the list. Choose a name for your current setup. Access your saved layouts later through the same menu. This helps if you switch between different types of work, like writing or editing photos.

## 🛠️ Troubleshooting Common Issues

Read these suggestions if the program stops working or behaves in ways you did not expect.

### The Program Does Not Start
Check if your antivirus software blocked the file. Open your security settings to permit the application. Restart the computer if the issue persists.

### Windows Do Not Snap
Ensure you have the latest drivers for your graphics card. Update your Windows version through the System Settings menu. Sometimes, old updates prevent windows from moving smoothly.

### Icons Look Small 
Go to the Display settings in Windows. Scale your text and apps to 100 percent if you use a high-resolution monitor. This fixes alignment issues with the snap zones.

## 💻 Customizing Settings

Modify how the program acts to suit your workflow. Open the Settings menu to view your options.

- Start at Login: Keep this checked if you want the program to load when you turn on your machine.
- Snap Strength: Change this value to make windows feel heavier or lighter when they snap to the grid.
- Keyboard Shortcuts: Assign keys to move windows without your mouse. Click a command line, then press your chosen key combination.
- Grid Density: Adjust the number of lines on your infinite canvas. More lines allow for smaller, more precise window placements.
